# GitHub Final Project – Simple Interest Calculator

This repository contains a simple interest calculator created as part of the GitHub Final Project.

## Description
The simple interest calculator is a Bash script that calculates simple interest based on user input.

## Formula
Simple Interest is calculated using the formula:

Simple Interest = (Principal × Rate × Time) / 100

## How to Use
1. Run the script `simple-interest.sh`
2. Enter the principal amount
3. Enter the rate of interest
4. Enter the time period in years
5. The script will display the calculated simple interest
